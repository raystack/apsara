# Apsara 2.0

<br/>
Apsara is an elegant and beautiful re-usable React UI components build using Radix UI and Stitches CSS-in-JS.
With Apsara, you can easily create clean and powerful personal sites, or quickly build modern web applications based on highly customizable design system.
