## Features

- Flexible positioning on either side of the page
- Customizable header with title, icon, and actions
- Sections for organizing content

## Usage

The SidePanel component provides a way to show data in the side of the page.

## Props

### SidePanel Props
- `side`: The side to position the panel. Can be 'left' or 'right'.
- `children`: Content to render inside the side panel.
- `className`: Additional CSS class names to apply.

### SidePanel.Header Props
- `title`: The title text to display in the header.
- `description`: Optional description text below the title.
- `icon`: Optional icon element to display before the title.
- `actions`: Array of action elements to display in the header.
