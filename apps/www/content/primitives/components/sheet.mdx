## Usage

The Sheet component provides a way to show complementary content in a sliding panel. It supports different positions, accessibility features, and customizable content.

## API Reference

### Sheet Props

- `children`: The content to be rendered
- `ariaLabel`: Accessible label for the sheet trigger

### Sheet.Content Props

- `side`: The direction from which the sheet appears (`"top" | "right" | "bottom" | "left"`)
- `close`: Whether to show the close button
- `ariaLabel`: Accessible label for the sheet content
- `ariaDescription`: Accessible description for the sheet content
- `children`: The content to be rendered inside the sheet

### Sheet.Title Props

- Inherits all HTML heading element props

### Sheet.Description Props

- Inherits all HTML paragraph element props
