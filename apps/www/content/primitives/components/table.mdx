## Props

Customize the Table appearance with these configuration options.

The `Table` component is composed of several parts, each with their own props:

### Table

The root container for the table.

- `className`: Additional CSS class names

### Table.Header

Defines the tableâ€™s header row.

- `className`: Additional CSS class names

### Table.Body

Wraps the table rows inside the body section.

- `className`: Additional CSS class names

### Table.Row

Represents a single row in the table.

- `className`: Additional CSS class names

### Table.Head

Represents a header cell inside the `Table.Header`

### Table.Cell

Represents a single cell in a row

- `className`: Additional CSS class names

### Table.SectionHeader

Represents a section heading, grouping different rows of the table.

- `classNames`: Map of className with internal components
  - `row`
  - `cell`
- `colSpan`: number of cells of the row. it ensures the section header spans across all columns of the table
