## Usage

The Overlay component provides a semi-transparent layer that dims the background content when dialogs or modals are active.

## Installation

Install the component from your command line.

## Accessibility

The Overlay component follows WCAG guidelines:

- Automatically manages focus when dialog opens
- Includes proper ARIA attributes
- Respects reduced motion preferences
