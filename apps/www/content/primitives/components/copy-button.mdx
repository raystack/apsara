## Props

The `CopyButton` component extends `IconButton` props and accepts the following additional props:

- `text`: Text to be copied to clipboard (required)
- `resetTimeout`: Time in milliseconds before reverting to default icon (default: 1000)
- `resetIcon`: Whether to reset to default icon after copying (default: true)
- All IconButton props are supported
