## Props

### Select.Trigger

- `size`: Defines the size of the trigger (`"small"` | `"medium"`, default: `"medium"`)
- `variant`: Visual style variant (`"default"` | `"filter"`, default: `"default"`)
- `iconProps`: Props for the chevron icon
- `aria-label`: Accessible label
- `aria-describedby`: ID of element describing the select
- `aria-required`: Whether the select is required
- `aria-invalid`: Whether the select has an invalid value

### Select.Content

- `position`: Position of the content (`"popper"` by default)
- `className`: Additional CSS classes

### Select.Item

- `value`: The value of the item
- `textProps`: Props passed to the Text component
- `className`: Additional CSS classes

## Accessibility

The Select component follows WAI-ARIA guidelines:

- Trigger has role `combobox`
- Content has role `listbox`
- Items have role `option`
- ARIA labels and descriptions
