## Anatomy

Import and assemble the component:

```tsx
import { FilterCard } from '@raystack/apsara'

<FilterCard>
  <FilterCard.Section>
    <FilterCard.Item label="Label">
      Action content
    </FilterCard.Item>
  </FilterCard.Section>
  <FilterCard.Section title="Section Title">
    Content
  </FilterCard.Section>
  <FilterCard.Footer>
    Footer content
  </FilterCard.Footer>
</FilterCard>
```
