## Anatomy

Import and assemble the component:

```tsx
import { InputField } from "@raystack/apsara";

<InputField placeholder="Enter text" />
```

For labels, description, and error messages, wrap with [Field](/docs/components/field):

```tsx
import { Field, InputField } from "@raystack/apsara";

<Field label="Email" description="We won't share it" error={errors.email?.message}>
  <InputField placeholder="Enter email" />
</Field>
```

## API Reference

Renders a text input control. For label, description, and error support, use with [Field](/docs/components/field).

## Accessibility

- Use with [Field](/docs/components/field) for automatic label association and error linking
- Required state is communicated via `aria-required`
- Supports all native `<input>` attributes
