## Anatomy

Import and assemble the component:

```tsx
import { TextArea } from "@raystack/apsara";

<TextArea placeholder="Enter text" />
```

For labels, description, and error messages, wrap with [Field](/docs/components/field):

```tsx
import { Field, TextArea } from "@raystack/apsara";

<Field label="Bio" description="Tell us about yourself" error={errors.bio?.message}>
  <TextArea placeholder="Write something..." />
</Field>
```

## API Reference

## Accessibility

- Use with [Field](/docs/components/field) for automatic label association and error linking
- Required state is communicated via `aria-required`
- Invalid state is communicated via `aria-invalid`
- Content is scrollable when text exceeds the visible rows
